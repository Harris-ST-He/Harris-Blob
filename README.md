# Harris Blob
## 关于我

大家好，我是 Harris。

- 普通双非26届
- 计算机科学与技术专业
- 暂时一段小厂实习经历，也还没有正式工作
- 前端略懂一些，但仍在持续学习和摸索

我常常会被 AI 时代的变化搞得有些不知所措。
一方面会觉得它来得太快，很多东西还没真正学稳；另一方面又忍不住去想，前端会不会真的慢慢失去位置。

但至少现在，我还是想继续认真学下去，继续写下去，也继续做网页。
我不希望前端死掉。对我来说，它依然是表达、连接内容与体验、把想法变成可见之物的一种方式，也是整个计算机领域最让我有兴趣的方向。
